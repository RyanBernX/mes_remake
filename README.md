# RB_Mes_Remake
Fuki 式对话加强，6R 重制版。

## 用法

在 RMXP 的脚本编辑器的 `Main` 组之前新建一个脚本，将 `mes_remake.rb`
中的内容直接复制到脚本中即可使用。使用前请仔细阅读脚本前面的说明和配置区域。

## 范例工程

在[发布页](https://github.com/RyanBernX/mes_remake/releases)可以下载到完整的范例工程，制作时使用的工具为 RPGXP 1.05。
